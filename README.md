# Product Program Info

腾讯地图·探索社区 PM 实习期间的项目作品集。每个项目以单页 HTML 的形式组织，附真实界面截图与项目背景。

## 目录

- [`小红书-好内容-onepage/`](./小红书-好内容-onepage/) — 「好内容-创意灵感」产品 0→1。从博主痛点（找灵感 / 看样板 / 发后复盘）切入，介绍产品形态、关键判断与一期数据。

## 本地查看

每个项目目录下打开 `index.html` 即可。截图资源统一放在 `assets/` 子目录内，引用全部为相对路径，可离线浏览。

## 写作约定

- 内容来自真实工作产出，敏感数据（用户量、具体营收等）已脱敏或使用占比口径
- 时间、人名、产品代号以脱敏版呈现
- 截图来自探索社区 B 端真实界面
